Подробное описание

Модуль пользовательского интерфейса.

Этот модуль предоставляет адаптивный интерфейс для различных разрешений экрана, поддержку многомониторных конфигураций, настраиваемые элементы HUD, интерактивные обучающие подсказки, встроенный чат и голосовую связь, а также интеграцию со Steam Overlay.
